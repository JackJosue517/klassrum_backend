COMMENCER

                API Endpoint

                https://klassrum.vercel.app/api/

l'API Klassrum fournit un accès dynamique et flexible aux données pour construire une plateforme éducationnelle robuste. Vous pourrez avoir la liste des enseignants, étudiants, sessions de cours programmées, etc.

Pour utiliser cet API, vous aurez besoin d'une clé API. Veuillez nous contacter à jackjosue517@gmail.com pour avoir votre clé API.

enseignants


# Voici un exemple avec curl
curl \
-X GET https://klassrum.vercel.app/api/teachers/

Pour avoir la liste de tous les enseignants disponibles, lancer une requête GET :
https://klassrum.vercel.app/api/teachers/ 



Exemple de résultat :

Pour ajouter un nouveau enseignant, utiliser une requête POST :
https://klassrum.vercel.app/api/teachers/


PARAMETRES DE LA REQUÊTE

Champ Type Description
firstname String Prénom(s) de l'enseignant
lastname String Nom de famille de l'enseignant
role String Statut de l'enseignant au sein de la structure
courses Array Unité(s) d'enseignement(s) dispensé(s) par l'enseignant
fullname String (optionnel) - Nom complet de l'enseignant précédé de la mention "M."
_id String (Non requis) Auto-genéré

Pour supprimer un enseignant, lancer une requête DELETE :
https://klassrum.vercel.app/api/teachers/:id


PARAMETRES DE LA REQUÊTE

Champ Type Description
_id String Identifiant personnel de l'enseignant

Pour mettre à jour le statut d'un enseignant, utiliser une requête PUT :
https://klassrum.vercel.app/api/teachers/:id


PARAMETRES DE LA REQUÊTE

Champ Type Description
firstname String Prénom(s) de l'enseignant
lastname String Nom de famille de l'enseignant
role String Statut de l'enseignant au sein de la structure
courses Array Unité(s) d'enseignement(s) dispensé(s) par l'enseignant
fullname String (optionnel) - Nom complet de l'enseignant précédé de la mention "M."

Etudiants


# Voici un exemple avec curl
curl \
-X GET https://klassrum.vercel.app/api/students/

Pour avoir la liste de tous les étudiants inscrits, lancer une requête GET :
https://klassrum.vercel.app/api/students/

PARAMETRES DE LA REQUÊTE

Champ Type Description
secret_key String Votre clé d'API
genre String (optionnel) Une chaîne de caractères pour filtrer les enseignants par genre:
m: masculin
f: féminin
limit Entier (optionnel - par défaut: 10) La limite de nombres d'objets à retourner, compris entre 1 et 100.

Erreurs

l'API de Klassrum utilise les codes d'erreur suivants:

Code d'erreur Signification
X000 Quelques paramètres sont manquants. Cette erreur apparît quand vous oubliez de passer certains paramètres essentiels à la requête.
X001 secret_key inconnu ou invalide. Cette erreur apparaît si vous utilisez une clé d'API inconnu, invalide ou ayant expiré.
X002 secret_key invalide pour ce domaine. Cette erreur apparaît si vous utilisez une clé d'API non-spécifié à votre domaine donné.
X003 Token utilisateur inconnu ou invalide. Cette erreur apparaît si vous utilisez token utilisateur inconnu ou si le token utilisateur a expiré.
