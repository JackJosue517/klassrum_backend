Pour avoir la liste de tous les étudiants inscrits, lancer une requête GET :
https://klassrum.onrender.com/api/students/ 



    Exemple de résultat :

    {
    [
    {
        title: 'Nouvelle Notification',
        content: 'M. HOETOWOU Yaovi a ajouté une nouvelle session de cours.',
        type: 'NotificationType.NEW'
    },
    ]
    }

PARAMETRES DE LA REQUÊTE

Champ Type Description
title String Titre de la notification
date Date (optionnel et auto-generée) La date a laquelle l'action a été effectuée
type String (optionnel) Type de notification(NEW?, UPDATE?, DELETE?)
